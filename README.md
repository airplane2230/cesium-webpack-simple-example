# cesium-webpack-simple-example

+ webpack을 활용한 cesium+osm 단순 예제입니다.
+ osm tile link 또는 직접 만든 tile의 link가 필요합니다.  
ex) http://your_ip/hot/{z}/{x}/{y}.png

+ 실행 순서  
--> ./src/index.js에서 tile link를 변경합니다.  
--> npm run build(output 경로는 dist입니다)  
--> npm start
